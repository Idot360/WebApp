Our Mission

The NJC Table Tennis team aims to inculcate in our members valuable life skills and sound values of resilience and confidence, through the physical and mental challenges of sports. Fourier Series

The Fourier series is used to express a periodic function as a sum of sine and cosine functions.

For a function \(f(x)\) defined on \([-L, L]\), the Fourier series is given by:

\(f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left( a_n \cos \left(\frac{n\pi x}{L}\right) + b_n \sin \left(\frac{n\pi x}{L}\right) \right)\)

where:

  • \(a_0 = \frac{1}{L} \int_{-L}^{L} f(x) \, dx\)
  • \(a_n = \frac{1}{L} \int_{-L}^{L} f(x) \cos\left(\frac{n\pi x}{L}\right) dx\)
  • \(b_n = \frac{1}{L} \int_{-L}^{L} f(x) \sin\left(\frac{n\pi x}{L}\right) dx\)

Example: Fourier Series of a Square Wave

Consider a square wave \(f(x)\) defined on \([- \pi, \pi]\):

\[ f(x) = \begin{cases} 1 & \text{if } -\pi < x < 0 \\ -1 & \text{if } 0 < x < \pi \end{cases} \]

The Fourier series for this square wave is:

\(f(x) = \frac{4}{\pi} \sum_{n=1,3,5,\ldots}^{\infty} \frac{1}{n} \sin(nx)\)

Multivariable Calculus

Multivariable calculus deals with functions of multiple variables and includes concepts like partial derivatives, gradients, and multiple integrals.

Partial Derivatives

If \(f(x, y)\) is a function of two variables, the partial derivatives are:

\(\frac{\partial f}{\partial x} = \lim_{h \to 0} \frac{f(x+h, y) - f(x, y)}{h}\)

\(\frac{\partial f}{\partial y} = \lim_{h \to 0} \frac{f(x, y+h) - f(x, y)}{h}\)

Gradient

The gradient of a scalar function \(f(x, y, z)\) is a vector that points in the direction of the steepest ascent:

\(\nabla f = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z} \right)\)

Example: Gradient of a Function

For the function \(f(x, y) = 3x^2 + 4y^2\), the gradient is:

\(\nabla f = (6x, 8y)\)
